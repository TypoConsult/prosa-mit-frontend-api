Or you can follow the steps below to clone the repository and run it locally.

### 1. Clone this repository

Using the GitHub CLI:

```bash
gh repo clone pucm/nextra-docs-starter
```

### 2. Install

```bash
pnpm i
```

### 3. Run the development server

```bash
pnpm dev
```
