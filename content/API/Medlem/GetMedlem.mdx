# GetMedlem

GetMedlem is used to retrieve the information of a user.

---

<RequestBadge method="POST">api/Medlem/GetMedlem</RequestBadge>

### Body Parameters

| Name   | Description           | Required | Type   |
| ------ | --------------------- | -------- | ------ |
| apiKey | Api key               | True     | String |
| email  | The email of the user | True     | String |

#### Example

```JSON
{
  "apiKey": "xxxxxx-xxxxxx-xxxxxx",
  "email": "test@example.com"
}
```

### Returns

| Name    | Description                                  | Type                        |
| ------- | -------------------------------------------- | --------------------------- |
| status  | The status code (e.g. "success" or "error")  | String                      |
| data    | Object of all the information about the user | Collection of [Data](#data) |
| message | Status message                               | String                      |
