GetMedlem

GetMedlem is used to retrieve the information of a user.

POSTapi/Medlem/GetMedlem

Body Parameters

NameDescriptionRequiredType
apiKeyApi keyTrueString
emailThe email of the userTrueString

Example

{
  "apiKey": "xxxxxx-xxxxxx-xxxxxx",
  "email": "test@example.com"
}
