Webformular Fields

Field Object

NameDescriptionRequiredType
NameThe key / identifier of the fieldTrueString
TypeThe type of fieldfalseString
DataThe data within the fieldTrueCollection of FieldData

FieldData Object

NameDescriptionRequiredType
ValueThe value of the field itselfTrueString
TextUnknown (Required, but can be empty)TrueString

Example

{
  "Name": "email",
  "Data": [
    {
      "Value": "test@example.com",
      "Text": ""
    }
  ]
}

Many fields, like fastnet and adresse2 that are found in the response of GetMedlem can’t be set using this form. Other things, like the user’s CPR number is set as cpr in this case, but respnds with cprnr in GetMedlem

Example function for setting fields (Typescript)

const setValue = (key: string, value: string | number | boolean) => {
  const output = {
    Name: key,
    Data: [
      {
        Value: value,
        Text: ""
      }
    ]
  };
 
  return output;
};

Usage

const response = await client.post("/api/Webformular/SubmitWebformular", {
  ...
  Fields: [
    setValue("email", "test@example.com"),
    setValue("cpr", "9999999999"),
  ]
})
